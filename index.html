高频问题整理
0.怎样导入工程?
在工程管理页面下,选择需要导出(备份)的工程,点击导出按钮,在弹出的窗口中选择保存的位置,之后可以发给其他小伙伴导入使用.
1.导入工程后卡死?
win下删除类似路径下文件夹 C:\Users\Master\AppData\Roaming\datasmark
mac下删除类似路径下文件夹 /Users/Master/Library/Application Support/datasmark
2.如何高效使用工具?
在工作台中,使用 [方向键] 上下翻页. [CTRL+回车键] 提交完成.再次点击已标注文本将执行删除标注,在工作台右侧可以批量清空标注,也可单独删除标注.
3.如何自定义标注类型?
切换到标注类型管理页面,点击新增按钮,按照一定的格式要求填写/插入内容.新增标记需要 点击红色新增按钮 才能正确添加上.点击列表左侧箭头,可以展开预览页面.
4.新增项目没有标注类型选择?
首次打开需要初始化本地数据库,如果没有标注类型,可以尝试 切换一下页面 (如切换到项目管理页面),或重启应用试试.
5.怎样使用自己的项目
在工作台或者项目管理页面,新增项目后再对话框中点击 确定按钮 ,激活项目.如果不小心点击了取消,可以切换到项目管理页面中,找到刚刚创建的项目,点击标题旁边的 激活按钮 即可激活成功,切换回工作台页面将使用您自己的数据展开工作.
6.标注进度如何计算?
标注进度 = ( 已标注 + 无法标注 ) / 总数 (注意:标注工作完成后,请点击完成按钮或者快捷键[CTRL+回车],否则 无法计算进度 )
7.是否开源?
源码托管地址: https://github.com/cuiwang/DatasMark ,本应用也是基于开源代码编写而成.目前出于测试阶段,代码还未优化,,版本间可能变化较大,请谨慎使用.
8.工具出现问题,数据无法导出怎么办?
热心网友提供小技巧: win下将软件数据存储目录( C:\Users\Master\AppData\Roaming\datasmark )移动到其他"电脑"类似目录下,实现手动切换"底层"数据,再进行导出.
mac下同理( /Users/Master/Library/Application Support/datasmark ).
9.如何提交需求和Bug?
微信添加 " heiscuiwang ",作者将免费,优先处理您的请求.
10.如何分组?
定义一个 Group 开头的标签，在标记完实体以后，将整句再标记为Group*.这样虽然Group和Entity是平级，但是可以通过position判断出哪些Entity是在Group中。
